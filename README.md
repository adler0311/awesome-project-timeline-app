## Awesome Project Timeline App

프로젝트 타임라인을 관리하기 위한 앱입니다.

---

## 기술 스택

- react
- react-native
- react-hook
- typescript
- firebase
- mobx

---

## 시작하기

1. npm install
2. yarn android or yarn ios
